# neural_network_basic
A basic neural network, more complex version will be added slowly

Version 1.0 : A binary classification model based on a scatter plot of 12 data points, where the data is separated into two classes using a linear decision boundary (line of best fit) with a negative slope.

Version 1.1 : A binary classification model based on a scatter plot of 12 data points, where the data is separated into two classes using a nonlinear decision boundary defined by a parabolic curve, enabling more accurate classification of nonlinearly separable patterns.

Version 1.2 : A classification model that uses a hidden layer to separate 12 data points into two groups. Instead of using a single equation, the model combines multiple simple lines created by neurons to form a more complex decision boundary.
